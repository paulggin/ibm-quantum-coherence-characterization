# ibm-quantum-coherence-characterization
Characterizes qubits via T1, T2, and randomized benchmarking on IBM hardware. Phase 1: single-qubit and full 156-qubit characterization on FakeMarrakesh. Phase 2: validate results on a 4-qubit real IBM backend. Pipeline includes circuit generation, transpilation, execution, decay fitting, outlier handling, and chip-scale visualization.
